## Visual Tone
- Clinical, calm, and academic
- Minimal color palette (white, light greys, muted blues)
- Typography prioritizing readability and hierarchy

## Layout Principles
- Strong information architecture
- Consistent sectioning across pages
- Prominent citations and evidence markers

## Landing Page Structure
1. Header
   - Logo (HairCare.ai)
   - Primary navigation: Conditions, Treatments, Science, About

2. Hero Section
   - Clear value proposition: "Evidence-based knowledge on hair and scalp health"
   - Subtext emphasizing neutrality and medical grounding

3. Trust Signals
   - Evidence-based badge
   - Medical disclaimer snippet
   - Citation count / guideline references

4. Core Topics Overview
   - Grid of major categories (Hair Loss, Scalp Health, Hair Science)

5. How to Use the Site
   - Explanation of evidence levels
   - How content is structured

6. Footer
   - Full medical disclaimer
   - About / Methodology / Contact (non-commercial)

## Content Page Visual Structure
- Page title
- Summary box (key facts)
- Evidence level indicator
- Main content sections
- Inline citations
- References section
- Last reviewed date
